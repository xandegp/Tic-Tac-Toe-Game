# Tic-Tac-Toe-Game

This Project is a recreation of the Tic Tac Toe Game in the Program Language : C

There is a 3x3 board and 2 players,where the first player to put 3 marks in sequence (i.e 3 in a line, 3 in a column or 3 in any diagonal) wins!
if the board became without space and none of the players wins, its a draw

You can play until u say u dont want to play anymome( i.e you can play forever :D )

Have fun!
